                                  Репино Cronwell Park Отель  -  современный загородный отель для активного отдыха. Отель расположен в одном из самых живописных мест Карельского перешейка, на берегу Финского залива в поселке Репино Курортного района Санкт-Петербурга,  в 45 км. от города. В отеле 118 номеров различных категорий, есть конференц-залы для проведения мероприятий. Ресторан отеля на 200 мест. Также в отеле есть боулинг, бильярд, караоке, водно-оздоровительный центр.
