        Наши мероприятия
    
    

                
                
                    Рабочее совещание  СПб ГБУ «Конгрессно-выставочное бюро» с представителями образовательных учреждений Санкт-Петербурга, готовящих специалистов по направлению «туризм», «сервис», «индустрия гостеприимства»,  «конгрессно-выставочная деятельность»
                
                16 мая 2017 г.
            
        

        




    
        
            
                
                
            
                
                
            
                
                
            
        
    



        
            
                
                16 мая 2017 года в зале заседаний Учёного совета СПб Государственного Экономического университета состоялось рабочее совещание  СПб ГБУ «Конгрессно-выставочное бюро» с представителями образовательных учреждений Санкт-Петербурга, готовящих специалистов по направлению «туризм», «сервис», «индустрия гостеприимства»,  «конгрессно-выставочная деятельность».

В формате живой дискуссии обсуждались следующие вопросы:

Текущая ситуация по подготовке кадров в сфере туризма: 


	специализация; образовательные программы по отраслевым направлениям «туризм», «сервис», «индустрия гостеприимства»;
	профессиональные стандарты (утвержденные, на рассмотрении, в планах на разработку);
	выпускники, трудоустройство, запросы отрасли;
	курсы повышения квалификации, дополнительные образовательные программы, по профессиональным квалификациям.


О разработке профессионального стандарта «Руководитель проектов в сфере делового туризма/выставочно-конгрессной деятельности»: инициативы, подходы к разработке, создание рабочей группы по разработке документа.

Сертификация отраслевых специалистов:  целесообразность, потребность отрасли, возможности образовательных учреждений.

Участниками мероприятия выступили представители таких учебных учреждений как СПбГЭУ, СПбГУПТиД, НИУ ВШЭ СПб, БАТиП.

В ходе открытой дискуссии все намеченные вопросы были обсуждены: освещена текущая ситуация по подготовке кадров в сфере туризма, достигнуто общее понимание необходимости продолжения работы по разработке профессиональных стандартов, в том числе профстандарта по направлению «Руководитель проектов в сфере делового туризма/выставочно-конгрессной деятельности», обсужден вопрос о возможности и целесообразности сертификации отраслевых специалистов.

Участники встречи выразили пожелание сформировать рабочую группу из представителей заинтересованных образовательных учреждений для продолжения диалога и разработки необходимых профессиональных стандартов.
