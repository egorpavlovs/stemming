        Новости
    
    

            
            
                Премия ТОП-100 ресторанов Петербурга
            
            28 ноября 2018 г.
        

        






        
            
                
                30 ноября 2018 года во второй раз состоится награждение премии ТОП-100 Ресторанов Петербурга, проходящей при поддержке Комитета по развитию туризма Санкт-Петербурга. Церемония пройдет в ресторане Строганофф Стейк Хаус.

Для методики оценки ресторанов был использован опыт партнеров из Финляндии Viisi Tahtea, которые вот уже несколько лет проводят премию ТОП 50 ресторанов Финляндии и являются экспертами для таких престижных рейтингов как White Guide Nordic и The World’s 50 Best Restaurants.

Председатель Комитета по развитию туризма Санкт-Петербурга Евгений Панкевич прокомментировал:
«Премия «ТОП-100 ресторанов Петербурга» должна стать гидом по ресторанному миру Петербурга на 2019 года для гостей и жителей города».
